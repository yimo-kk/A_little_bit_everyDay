### react

##### 1.安装以及创建react+ts项目

###### 全局安装  

create-react-app npm i create-react-app -g 
/ yarn global add create-react-app 

###### 查看版本

 create-react-app --version 

###### 使用 typescript 模板创建项目 

npx create-react-app rtsdemo --template typescript

###### 运行 

cd rtsdemo
 npm start / yarn start
